# Sales Performance Dashboard

## Overview

This project presents an interactive Sales Performance Dashboard created using Microsoft Excel. The dashboard provides insights into overall sales performance across products, regions, and salespersons.

## Key Metrics

- Total Sales: ₹1,29,44,500
- Units Sold: 4,705
- Total Profit: ₹38,34,400
- Average Sales: ₹2,58,890

## Dashboard Features

- Product-wise sales analysis
- Region-wise sales analysis
- Salesperson performance comparison
- Units sold by product
- Interactive Product, Salesperson, and Region slicers
- Dynamic KPI cards that update based on filter selections

## Tools & Techniques Used

- Microsoft Excel
- PivotTables
- PivotCharts
- Slicers
- Calculated Fields
- Custom Number Formatting
- Data Visualization
- Interactive Dashboard Design

## Insights

- Sales performance varies significantly across products and salespersons.
- The dashboard allows users to dynamically analyze performance using interactive slicers.
- Product and regional comparisons help identify strong and weak areas of sales performance.

## Project Purpose

This project was created to strengthen my skills in Excel data analysis, PivotTables, data visualization, and interactive dashboard development.
